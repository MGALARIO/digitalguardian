Variables & Data Types

Variables are like containers with names where you can put your data.

Think of a variable as an empty box with a name, waiting for you to put something in it.

Data is what you put inside a variable. It is the content or value stored in the container.

To put data into a variable in Python, we use the equals sign (=). The box name goes on the left side, and the data goes on the right side.

name = "Michael"
age = 25
temperature = 36.5

Key Concepts:

  • Strings: Text enclosed in quotes ("hello" or 'world')
  • Integers: Whole numbers (42, -7, 0)
  • Floats: Decimal numbers (3.14, -0.5, 2.0)
  • Booleans: True or False values
