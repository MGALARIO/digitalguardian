Hardware Components
The physical, tangible parts of a computer system that you can see and touch. Hardware is the foundation upon which all software runs.
CPU — the brain that executes instructions
RAM — fast, temporary working memory
Storage — HDDs, SSDs, NVMe drives
Motherboard — the main interconnect board
I/O devices — keyboard, mouse, monitor
CPU Architecture
The Central Processing Unit consists of three main units: the Arithmetic Logic Unit (ALU) for calculations, the Control Unit (CU) for directing operations, and Registers for immediate data storage.

Computers use different number bases for different purposes. Understanding conversions between bases is essential for programming and digital systems.

Binary (Base-2): digits 0–1, computer use
Octal (Base-8): digits 0–7, compact binary
Hexadecimal (Base-16): 0–9 + A–F

Binary Addition Rules
0 + 0 = 0 (sum=0, carry=0) 0 + 1 = 1 (sum=1, carry=0) 1 + 0 = 1 (sum=1, carry=0) 1 + 1 = 10 (sum=0, carry=1) 1 + 1 + 1 = 11 (sum=1, carry=1) // Example: 13 + 10 = 23 1101 (13) + 1010 (10) ────── 10111 (23) ✓
Two's Complement
Method to represent negative numbers. To negate a number: invert all bits, then add 1.
// Represent -5 in 8-bit two's complement +5 = 00000101 Invert: 11111010 Add 1: 11111011 ← -5 in two's complement
TOPIC 05 History
🕰️
History of Computing
Computing has evolved through five generations over 80 years — from room-sized vacuum tube machines to AI-powered microprocessors that fit in your pocket.
1st Gen (1940s–50s): Vacuum tubes, ENIAC
2nd Gen (1950s–60s): Transistors replace tubes
3rd Gen (1960s–70s): Integrated circuits
4th Gen (1970s–now): Microprocessors, PCs
5th Gen (present): AI, quantum computing
Five Generations
1940s–50s
1st Gen: Vacuum tubes, punched cards, ENIAC (18,000 tubes), UNIVAC. Very large, hot, unreliable. Machine language only.
1950s–60s
2nd Gen: Transistors replace vacuum tubes. Smaller, cheaper, faster. Assembly language & early high-level languages (FORTRAN, COBOL).
1960s–70s
3rd Gen: Integrated Circuits (ICs). Multiple transistors on a single chip. IBM System/360. Operating systems introduced. C language.
1970s–now
4th Gen: Microprocessors (Intel 4004, 1971). Personal computers: Apple II, IBM PC. GUI introduced. Internet era. Moore's Law.
Present
5th Gen: AI & machine learning, quantum computing, natural language processing. Parallel processing, neural networks, edge computing.
