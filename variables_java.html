Variables in Java

Variables & Data Types

Variables are containers that hold data values in Java. Unlike Python, Java is statically typed, meaning you must declare the type of data a variable will hold.

Think of a variable as a labeled box that can only hold a specific type of item.

Data Types define what kind of data can be stored in a variable.

String name = "Michael";
int age = 25;
double temperature = 36.5;

Key Concepts:

  • String: Text enclosed in double quotes ("hello")
  • int: Whole numbers (42, -7, 0)
  • double: Decimal numbers (3.14, -0.5, 2.0)
  • boolean: true or false values
  • Static Typing: Variable types must be declared before use
