ALU Structure & Register Chaining — the missing piece between bits and programs

This visualizer shows the block-level structure of an ALU (not the gates inside) plus a register file sitting next to it. Pick a bit-width, enter up to three numbers, and press GO to watch the CPU add them using the ALU twice — saving the first result into a register, then feeding it back in. For gate-level detail, see index.html. For background theory, see foundations.md §11.
